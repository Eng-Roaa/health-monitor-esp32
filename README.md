# Smart Health Monitoring System (ESP32)

## Overview
This project is a smart biomedical monitoring system built using ESP32. It monitors heart rate, SpO2, fall detection, medicine reminders, and SOS emergency alerts.

## Features
- Heart Rate Monitoring (MAX30102)
- SpO2 Estimation
- Fall Detection (MPU6050)
- Real-Time Clock (RTC DS1302)
- Medicine Reminder System
- SOS Emergency Button
- Blynk IoT Mobile App Integration
- LCD Display (I2C 16x2)

## Hardware Used
- ESP32
- MAX30102 Pulse Oximeter
- MPU6050 IMU
- DS1302 RTC Module
- LCD 16x2 I2C
- Buzzer
- Push Button (SOS)

## IoT Platform
- Blynk Cloud
